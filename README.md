# esper.js

A low-level WebGL rendering framework.

### Installation

Requires [bower](http://bower.io/).

```bash
bower install esper
```

### Building

Requires [node](http://nodejs.org/).

To download dependencies:
```bash
npm install
```

To compile the project to `esper.js` and `esper.min.js` in the `build/` directory:
```bash
npm run build
```
